# Mahdi-Ataey-intro-26.2
My Weekly Assignment for Intro to Programming course with Code the Dream
Welcome to my repository for the Intro to Programming course. This project contains the solutions and exercises for the first four weeks of the program.

---

## Weekly Breakdown

### Week 1: Basics
**Concepts Covered:**
* Basic JavaScript Syntax and Variable Declaration (`let`).
* Data Types (Strings, Numbers, Booleans).
* Conditional Statements (`if/else`).
* Function declaration, parameter passing, and encapsulation.

### Week 2: Loops and Arrays
**Concepts Covered:**
* Iteration using `for` and `while` loops.
* Array manipulation and indexing.
* Solving logic problems with the Modulo operator.
* Practical methods: `slice()`, `splice()`, `push()`.

### Week 3: Objects
**Concepts Covered:**
* Object literals and property management.
* Using `for...in` loops to iterate over objects.
* Implementing methods and the `this` keyword.
* Object Constructors and instantiating instances (`new` keyword).
* Working with the built-in `Date` object.

### Week 4: Algorithms and Callbacks
**Concepts Covered:**
* Implementing mathematical algorithms (Temperature conversion, Tip calculation).
* String manipulation and reversal algorithms.
* Prime number detection and arrays.
* Student grading logic and calculation.
* Understanding and implementing Callback functions.

---

## Repository Structure
- `week1.js`: Fundamentals of JS syntax and conditionals.
- `week2.js`: Logic, loops, and data structures.
